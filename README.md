# aips-challenge

Python version 3.9

<span style="color:red">All commands to be executed from project root.</span>.

### Running the project
```shell
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python src/main.py
```

### Running tests
```shell
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python -m unittest discover . 
```
